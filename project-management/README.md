# Project Management Artifacts

This directory contains formal project management deliverables supporting Phase One of the AWS Cloud Governance Program.

These artifacts demonstrate real-world project coordination, scope control, and governance oversight practices. They are not academic templates and reflect execution within a constrained, audit-aware environment.

## Included Artifacts

- Project Charter
- Scope and Guardrails
- Implementation Plan
- Risk Register
- Decision Log
- Weekly Status Report (Sample)
- Phase Closeout Report
- Lessons Learned Summary

## Supporting Governance and Control Artifacts

- Communications Plan
- Stakeholder Register
- Evidence Index (Authoritative Mapping)

Artifacts are provided in read-only formats where applicable and contain no sensitive or proprietary information.
